# Insect Alphabet 🦋🔍

**Insect Alphabet** is an interactive, browser-based educational discovery tool designed to introduce early learners to the worlds of literacy and entomology simultaneously. Moving away from traditional "memorization-first" learning models, this framework leverages high-contrast visual cues, mnemonic anchors, and micro-interactions to foster semantic agency and natural cognitive development.

---

## 🧠 Core Design Pillars

This project applies a human-centric UI/UX approach to early childhood instruction:
* **Taxonomic Exploration:** Connects individual letters of the alphabet with distinct insect archetypes, encouraging early scientific reasoning and categorical sorting.
* **Mnemonic Anchoring:** Utilizes vibrant visual designs and structured layouts to act as psychological anchors, facilitating easier long-term memory retrieval.
* **Cognitive Scaffolding:** Features a clean, distraction-free grid system designed to minimize cognitive load, allowing young learners to focus entirely on associative learning.

---

## 🛠 Technical Stack & Metadata
* **Components:** Interactive Multi-Sensory Literacy Grid.
* **Tech Stack:** HTML5 | CSS3 | JavaScript.
* **Design Philosophy:** Replacing rote memorization with structural, semantic agency and associative learning.
* **Deployment:** Hosted live via GitHub Pages.
